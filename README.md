## Deploying API

1. Configure DB

  - Install MongoDB
  - Create a BD "buses-db"

2. Clone the project

```
git clone https://github.com/pedris11s/buses-server.git
```

3. Install dependencies

```
cd buses-server
npm install
```

4. Running the app

```
sails lift
```

Note: The user by default is "root" and pass "toor". You don't need to created.
