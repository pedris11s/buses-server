## Deploying API

1. Configure DB

  - Install MongoDB
  - Create a BD "buses-db"

2. Clone the project

```
git clone http://git.kimirik.com/tazpro/wsoft.buses-server.git
```

3. Install dependencies

```
npm install
```

4. Running the app

```
sails lift
```
